# Vantage - Decisions Upgraded
Vantage is a full-stack SaaS inventory and analytics platform designed to help businesses manage products, orders, users, companies, alerts, and operational insights from a single dashboard.

## Features
- Admin and client dashboards with role-based access
- JWT-based authentication and protected routes
- User and company management for admins
- Inventory management with stock status tracking
- Order tracking with revenue and status updates
- Real-time dashboard counter updates
- System and stock alert workflows
- Analytics views with charts and business metrics
- Responsive UI for desktop and mobile screens

## Tech Stack
- React.js
- Node.js
- Express.js
- PostgreSQL
- Tailwind CSS
